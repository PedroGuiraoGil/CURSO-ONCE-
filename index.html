Manual de Accesibilidad y Recursos Iniciales

Una guía de alto contraste para personas con discapacidad visual y los recursos clave para empezar.

1. Cómo acceder a la Configuración de Accesibilidad

Presiona Windows + U para abrir directamente la Configuración de Accesibilidad.

2. Lupa

Cómo activar/desactivar la Lupa
  • Activar: Presiona Windows + + (tecla más)
  • Desactivar: Presiona Windows + Esc
Modos de visualización de la Lupa

Presiona Ctrl + Alt + M para cambiar entre los modos:

  • Lente: Una ventana pequeña que sigue el cursor.
  • Pantalla completa: Amplía toda la pantalla.
  • Acoplado: Una sección fija muestra la ampliación.

3. Puntero del mouse

Cómo cambiar tamaño y color del puntero

Abre Configuración con Windows + U, luego ve a Cursor y puntero para ajustar el tamaño y color.

4. Narrador (lector de pantalla integrado)

Activar/Desactivar Narrador

Atajo: Ctrl + Windows + Enter

Atajos importantes:
  • Ctrl: Pausar/continuar lectura.
  • Caps Lock + M: Modo de navegación.
  • Caps Lock + H: Saltar a siguiente encabezado.

5. Otras opciones de accesibilidad útiles

  • Texto en negrita: Ve a Windows + U > Texto en pantalla.
  • Alto contraste: Actívalo con Alt + Shift + Impr Pant.
  • Teclado en pantalla: Actívalo con Windows + Ctrl + O.
  • Subtítulos en directo: Actívalo con Windows + Ctrl + L.

Destruye los mitos, entiende la magia negra de los LLMs y empieza a crear desde el minuto uno. La teoría mínima necesaria para ser peligroso con IA.

1. El Qué y el Porqué de la IA (La Única Teoría Necesaria)

🧠 ¿Qué es realmente la IA?

Olvida las definiciones académicas. La IA es simplemente esto: enseñar a las máquinas a encontrar patrones. En lugar de decirle a un ordenador "si ves un gato, es porque tiene bigotes, cuatro patas y maúlla", le mostramos 10,000 fotos de gatos y dejamos que él descubra qué hace que un gato sea un gato.

Es como enseñar a un niño, pero este niño puede procesar millones de ejemplos por segundo y nunca se cansa.

🎭 La Caja Negra: El Misterio de los LLMs

El concepto fundamental: Las IAs modernas son "cajas negras". Sabemos qué entra (tu prompt) y qué sale (la respuesta), pero el proceso interno es un misterio incluso para sus creadores.

Los números que marean:

  • GPT-3: 175 mil millones de parámetros
  • GPT-4: Se estima en 1.7 billones de parámetros
  • Modelos futuros: Apuntando a los 10 billones

Cada parámetro es como una neurona artificial. Para ponerlo en perspectiva: el cerebro humano tiene aproximadamente 86 mil millones de neuronas. ¡Los modelos de IA ya nos superan en cantidad!

2. Nivel Intermedio: El Arte y Ciencia del Prompting

🎯 La Fórmula Maestra del Prompting: ACTFOR

Olvida el básico "Actúa como...". Aquí está la fórmula completa que usan los pros:

  • Audience (Audiencia): ¿Para quién es esto?
  • Context (Contexto): ¿Cuál es la situación?
  • Task (Tarea): ¿Qué necesitas exactamente?
  • Format (Formato): ¿Cómo lo quieres presentado?
  • Objective (Objetivo): ¿Qué buscas lograr?
  • Restrictions (Restricciones): ¿Qué límites hay?

🔗 Técnicas Avanzadas de Prompting
1. Chain-of-Thought (Cadena de Pensamiento)

Fuerza a la IA a mostrar su razonamiento paso a paso:

Resuelve este problema mostrando tu razonamiento paso a paso: [Tu problema] Antes de darme la respuesta final, muéstrame cómo llegaste a ella.
2. Few-Shot Learning

Dale ejemplos de lo que quieres:

Convierte estas descripciones de producto en copies persuasivos: Ejemplo: Entrada: "Zapatos cómodos para caminar" Salida: "Zapatos que hacen que 10,000 pasos se sientan como caminar sobre nubes" Ahora convierte estos: 1. "Café orgánico de comercio justo" 2. "Curso online de marketing digital"
3. Role-Playing Avanzado

No solo asignes un rol, define la personalidad completa:

Eres Maya, una consultora de marca con 15 años de experiencia en Silicon Valley. Características: - Directa pero empática - Usas metáforas del mundo tech - Siempre das ejemplos de empresas reales - Odias el marketing fluff - Tu frase favorita es "Los datos no mienten" Con esta personalidad, analiza mi propuesta de marca: [tu propuesta]

Subtítulos en Vivo y Modo Concentración

  • Lector Inmersivo (Edge/Office): Mejora la legibilidad y comprensión de textos con herramientas como enfoque de línea y resaltado gramatical.

Navegación Web Accesible

Herramientas Multifuncionales: Microsoft PowerToys

Una Suite Completa para la Accesibilidad

PowerToys es una colección de utilidades de Microsoft que incluye varias herramientas clave para la accesibilidad:

  • Mouse Utilities: Incluye funciones para encontrar el cursor fácilmente, resaltar los clicks y mostrar una cruz sobre el puntero.

Prácticas: De 0 a 100 con Gemini

Un manual práctico para guiarte desde las interacciones más sencillas hasta las tareas más complejas, aprendiendo a través de la experimentación directa con la IA.

Manual de Prácticas de IA: De 0 a 100 con Gemini 🚀

¡Bienvenido a tu viaje por el mundo de la Inteligencia Artificial! Este manual te guiará paso a paso, desde las interacciones más sencillas hasta las tareas más complejas, utilizando Gemini. El objetivo es que aprendas haciendo, experimentando y, sobre todo, conversando con la IA.

Módulo 1: El Arte de la Conversación y la Búsqueda (Nivel 0-20)

Aquí empezarás a "hablar" con la IA. El objetivo es que te sientas cómodo y entiendas cómo formular preguntas para obtener los mejores resultados.

Práctica 1: ¡Hola, Mundo!

Objetivo: Romper el hielo.
Acción: Inicia una conversación simple.
Prompt de ejemplo: Hola, ¿quién eres y qué puedes hacer?
Explora: Pregúntale sobre el tiempo, pídele un chiste o una curiosidad. Acostúmbrate a su estilo de respuesta.

Práctica 2: El Buscador Inteligente

Objetivo: Realizar búsquedas de información.
Acción: Usa Gemini como si fuera un motor de búsqueda, pero pide resúmenes.
Prompt de ejemplo: ¿Quién fue Hipatia de Alejandría? Explícamelo como si tuviera 15 años.
Explora: Pide comparaciones (¿Qué diferencias hay entre un delfín y un marsopa?), listas (Hazme una lista de las 5 montañas más altas del mundo) y cronologías (Crea una línea de tiempo de la carrera espacial).

Práctica 3: El Poder del Contexto

Objetivo: Mantener una conversación con seguimiento. Gemini recuerda lo que has dicho antes en la misma conversación.
Acción: Haz una pregunta y luego haz preguntas de seguimiento.
Prompt de ejemplo:
Recomiéndame tres destinos para viajar en verano en España.
De esos tres, ¿cuál es el más económico?
Ok, y para ese destino, ¿qué actividades puedo hacer al aire libre?

Práctica 4: Adoptando una Personalidad

Objetivo: Aprender a darle un "rol" a la IA para que adapte sus respuestas.
Acción: Pídele que actúe como un personaje o experto.
Prompt de ejemplo: Actúa como un guía turístico de Roma. Descríbeme un día perfecto visitando la ciudad, incluyendo dónde comer la mejor pasta.
Explora: Pídele que sea un crítico de cine, un entrenador personal o un experto en vinos.

Módulo 2: Trabajando con Texto (Nivel 20-40)

Ahora que ya sabes conversar, vamos a profundizar en la creación y manipulación de texto.

Práctica 5: El Escritor Creativo

Objetivo: Generar diferentes tipos de texto creativo.
Acción: Pide poemas, guiones cortos o la letra de una canción.
Prompt de ejemplo: Escribe un poema corto sobre la lluvia en la ciudad con un tono melancólico.
Explora: Pide un diálogo entre dos personajes históricos, el inicio de una novela de misterio o un eslogan para una cafetería.

Práctica 6: El Asistente de Redacción

Objetivo: Mejorar y transformar tus propios textos.
Acción: Pega un texto y pide correcciones o cambios de estilo.
Prompt de ejemplo: Corrige la gramática y la puntuación de este párrafo: [Pega tu texto aquí]. Después, reescríbelo con un tono más formal.

Práctica 7: El Maestro del Resumen

Objetivo: Condensar información larga en puntos clave.
Acción: Pega un artículo largo o un texto complejo.
Prompt de ejemplo: Resume este artículo en 5 puntos clave: [Pega el texto del artículo].
Explora: Pide un resumen en una sola frase, o un resumen para una audiencia específica (ej. "para niños", "para expertos en el tema").

Módulo 3: El Mundo Visual - Imágenes y Archivos (Nivel 40-60)

Vamos a interactuar con elementos visuales y documentos.

Práctica 8: ¿Qué ves? - Análisis de Imágenes

Objetivo: Subir una imagen y obtener información sobre ella.
Acción: Sube una foto de un monumento, un plato de comida o una planta.
Prompt de ejemplo (subiendo una foto de la Sagrada Familia): ¿Qué es esto? Dame información sobre su historia y arquitecto.
Explora: Sube la foto de los ingredientes que tienes en la nevera y pregunta: ¿Qué puedo cocinar con esto?.

Práctica 9: De Imagen a Texto

Objetivo: Extraer y trabajar con el texto de una imagen.
Acción: Sube una foto de una página de un libro o de un cartel.
Prompt de ejemplo: Transcribe el texto de esta imagen. Luego, tradúcelo al inglés.

Práctica 10: El Analista de Documentos (PDF, Docs)

Objetivo: Subir un archivo y hacer preguntas sobre su contenido.
Acción: Sube un informe en PDF o un documento de Word.
Prompt de ejemplo (subiendo un informe de ventas): Basado en este informe, ¿cuál fue el mes con mayores ventas? Haz un resumen de las conclusiones principales.
Explora: Sube tu currículum y pide: Dame sugerencias para mejorar mi CV basándote en este documento.

Módulo 4: La Dimensión Sonora y Audiovisual - Voz y Vídeo (Nivel 60-80)

Ahora, la IA escuchará tu voz y "verá" vídeos.

Práctica 11: Conversación por Voz

Objetivo: Interactuar con Gemini usando tu voz.
Acción: Activa el modo de voz y realiza las prácticas de los módulos anteriores hablando en lugar de escribir.
Prompt de ejemplo (hablando): "Busca recetas de paella valenciana y léeme los ingredientes."
Explora: Mantén una conversación fluida por voz. Pídele que te cuente un cuento y usa diferentes tonos de voz para ver si influye en la respuesta.

Práctica 12: Analizando Vídeos

Objetivo: Obtener información a partir de un enlace de vídeo (por ejemplo, de YouTube).
Acción: Proporciona un enlace a un vídeo.
Prompt de ejemplo (con un enlace a una charla TED): Resume los puntos clave de este vídeo. ¿Cuál es el mensaje principal del ponente?
Explora: Pide una transcripción de los primeros minutos, o pregunta por momentos específicos del vídeo (¿En qué minuto habla sobre el futuro de la tecnología?).

Módulo 5: El Lenguaje de las Máquinas - Código (Nivel 80-100)

Llegamos al nivel más avanzado: usar Gemini como una herramienta de programación.

Práctica 13: El Generador de Código

Objetivo: Crear fragmentos de código en diferentes lenguajes.
Acción: Pide una función simple en un lenguaje de programación.
Prompt de ejemplo: Escribe una función en Python que reciba una lista de números y devuelva solo los números pares.

Práctica 14: El Depurador (Debugger)

Objetivo: Encontrar y corregir errores en el código.
Acción: Pega un código que no funciona y pide ayuda.
Prompt de ejemplo: Este código en JavaScript debería mostrar una alerta, pero no funciona. ¿Puedes encontrar el error? [Pega tu código con errores].

Práctica 15: El Traductor de Lenguajes

Objetivo: Convertir código de un lenguaje de programación a otro.
Acción: Proporciona un fragmento de código y pide su equivalente en otro lenguaje.
Prompt de ejemplo: Convierte esta función de Python a Java: [Pega la función en Python].

Práctica 16: El Explicador de Código

Objetivo: Entender código complejo.
Acción: Pega un código y pide una explicación.
